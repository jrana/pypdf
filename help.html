Getting Started

PyPDF is a powerful PDF viewer and editor that allows you to view, edit, and manage PDF documents with ease. Here's how to get started:

Opening a PDF

There are several ways to open a PDF file:

  • Click üìÇOpen in the toolbar
  • Use the menu: File ‚Üí Open
  • Press Ctrl+O
  • Drag and drop a PDF file onto the application window

Opening a Folder

You can also open an entire folder of PDFs:

  • Click üìÅOpen Folder in the toolbar
  • All PDF files in the folder will appear in the sidebar for easy access
üí° Tip: You can have multiple PDFs open at once. Each PDF opens in its own tab for easy switching.

Viewing PDFs

PyPDF offers two viewing modes to suit your needs:

üìÑSingle View (Continuous Scroll)

All pages are displayed vertically in a continuous scroll. Perfect for reading documents naturally from top to bottom.

üìëGrid View

See all pages as thumbnails in a grid. Great for getting an overview, reordering pages, or selecting multiple pages for editing.

Switching Views

  • Click üìÑSingle or üìëGrid in the toolbar
  • Use keyboard shortcuts: Ctrl+3 for Single, Ctrl+4 for Grid

Grid View Features

In Grid View, you can:

  • Select pages by clicking on them (checkbox appears)
  • Reorder pages by dragging and dropping thumbnails
  • Double-click a thumbnail to jump to that page in Single View
  • Select all or Deselect all pages using the toolbar buttons

Navigation

Navigate through your PDF with ease:

Page Navigation

  • Previous/Next Page: Click the ‚óÄ and ‚ñ∂ buttons in the toolbar
  • Go to specific page: Enter a page number in the page spinner
  • Scroll: In Single View, simply scroll to move between pages

Page Indicator

The current page is always shown in the toolbar. In continuous scroll mode, the page indicator automatically updates as you scroll through the document.

üí° Tip: The page indicator shows your current position as "Page X / Total". You can click on it and type a specific page number to jump directly there.

Inserting Pages

Insert Pages from Another PDF

  1. Click üì•Insert PDF in the toolbar or use Edit ‚Üí Insert PDF Pages
  2. Select a PDF file to insert from
  3. The Insert Dialog opens showing thumbnails of all pages
  4. Choose your selection mode:
    • üìÑ All Pages: Insert all pages from the source PDF
    • üñ±Ô∏è Select Pages: Click on thumbnails to select specific pages
  5. Choose where to insert (after which page number)
  6. Click Insert to add the pages
üí° Tip: When selecting pages, you can pick non-contiguous pages (e.g., pages 1, 3, and 7). They will be inserted in order.

Insert Images as Pages

  1. Click üñºÔ∏èInsert Image or use Edit ‚Üí Insert Images as Pages
  2. Select one or more image files (PNG, JPG, GIF, BMP, TIFF, WebP)
  3. Choose where to insert the images
  4. Each image becomes a new PDF page

Splitting & Extracting Pages

Extract Selected Pages

  1. Switch to Grid View
  2. Click on the pages you want to extract (or use Select All)
  3. Click ‚úÇÔ∏èExtract in the toolbar
  4. Configure the split options in the dialog
  5. Choose a location to save the new PDF

Delete Pages

  1. Switch to Grid View
  2. Select the pages you want to delete
  3. Click üóëÔ∏èDelete in the toolbar
  4. Confirm the deletion
‚ö†Ô∏è Warning: You cannot delete all pages. At least one page must remain in the document.

Saving & Printing

Saving Your PDF

  • Save: Press Ctrl+S or click the Save button to save changes to the original file
  • Save As: Press Ctrl+Shift+S to save to a new file

Unsaved Changes Warning

If you try to close the application with unsaved changes, PyPDF will warn you and give you the option to:

  • Quit Without Saving: Discard all unsaved changes
  • Cancel: Go back and save your work

Printing

  1. Click üñ®Ô∏èPrint or press Ctrl+P
  2. Configure your print settings in the system dialog
  3. Click Print to send to your printer
